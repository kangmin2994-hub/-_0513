# AI 인터뷰 정리

> 인터뷰 일자: 2026-05-13

## 1. 직업 — 프로젝트 매니저

**한 줄 요약:** 없던 것이 생겨나 잘 돌아가는 순간에 보람을 느끼는 PM — 다만 잘게 나뉜 이해관계자 조율과 일정관리가 가장 큰 도전이다.

| 질문 | 답변 |
|------|------|
| 지금 하고 있는 일이 뭔지 간단하게 얘기해줄 수 있어요? | 프로젝트 관리 매니저야. |
| 어떤 분야의 프로젝트를 주로 맡고 있어요? | 채널 쪽도 있고, 기간계 또는 신규사업 도입에 대한 프로젝트, SI성 프로젝트가 대부분이지. |
| 그 중에서 가장 재미있거나 보람을 느끼는 유형은 어떤 건가요? | 새로운 시스템을 도입하거나 비즈니스가 만들어져서 서비스가 잘 운영될 때 보람을 느끼지. |
| 기억에 남는 프로젝트가 있어요? | 가장 최근에 한 프로젝트는 카드꾸미기 프로젝트라고, 에디터를 통해서 내가 꾸미고 싶은 디자인으로 카드 플레이트를 만드는 프로젝트였어. |
| 그 프로젝트에서 "잘 됐다"고 느낀 순간이 구체적으로 언제였어요? | 오류 없이 이미지가 잘 만들어질 때. |
| 그 프로젝트 진행하면서 가장 어려웠던 부분은 뭐였어요? | 여러 이해관계자들과 협업하고 대응 개발이 필요해서 일정관리가 힘들었어. |
| 특히 어떤 이해관계자 사이에서 가장 부딪히는 경우가 많았어요? | 채널 쪽 담당자가 세부 업무별로 나눠져 있어서 힘들었어. |
| 그럴 때 본인만의 해결 방식이 있었나요? | *(미답변)* |

## 2. AI 활용 기대와 소비습관

**한 줄 요약:** 의지보다 구조가 필요한 사람 — 알고 있어도 잊어버리기 때문에, AI가 반복적으로 상기시켜주는 시스템 역할을 해주길 기대한다.

| 질문 | 답변 |
|------|------|
| 이 인터뷰를 시작하게 된 계기가 뭔가요? | 교육에서 하라고 해서 하는 거야. |
| 하라고 해서 시작했지만, 막상 해보니 의미 있을 것 같다는 느낌이 드나요? | 의미가 있는 것 같은데 아직 미숙해서 실제 사용을 잘할지는 모르겠어. |
| 어떤 면에서 의미가 있을 것 같다고 느꼈어요? | 잘 쓰면 업무나 실생활에서 많은 도움이 될 것 같아. |
| 특히 "이런 상황에서 AI가 있었으면 좋겠다"고 느낀 순간이 있었나요? | 여행 일정을 짜거나 카드 사용이나 매월 가계부를 자동으로 작성해서 절약해야 될 부분을 정리해서 알려줬으면 좋겠어. |
| 가계부 쪽에서 특히 어떤 부분이 신경 쓰이나요? | 습관적인 커피 비용이나 너무 과도한 구독 서비스 비용. |
| 알면서도 계속 유지하게 되는 이유가 있나요? | 귀찮아서 해지를 못해서. 커피는 캡슐로 내려마셔도 되는데 꼭 비싼 거 사먹는 습관이 있는 것 같아. |
| 비싼 커피를 사먹을 때, 맛 때문인가요 아니면 다른 이유가 있나요? | 가까이 있어서. 그리고 가끔 내가 사야 될 때는 저가형보다는 브랜드 커피를 마시는 경향이 있어. |
| 누군가한테 살 때 저가형이 꺼려지는 편인가요? | 맞아. |
| 그게 본인 지출에도 영향을 미치는 것 같아요? | 커피 비용이 월 10만 원 이상 지출되고 있어. 매월 소비금액 정리하다 보면 이렇게 많이 마셨나 싶을 때가 있어. |
| 스스로 인식하는 것만으로는 부족하고, 외부에서 계속 상기시켜줘야 한다는 느낌인가요? | 맞아. 계속 상기시켜주면 좋지. |

---

## 종합 인사이트

- **직업:** SI/채널 분야 PM. 새 시스템이 안정적으로 가동될 때 보람을 느끼며, 다수의 이해관계자 조율과 일정관리가 주요 과제.
- **AI 기대:** 가계부 자동화, 여행 일정 생성 등 반복적인 정보 정리와 지출 경고 역할을 AI에게 기대. 의지보다 시스템적 넛지(알림)를 선호.
