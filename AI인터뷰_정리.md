## 1. 職業 — プロジェクトマネージャー

**一言まとめ:** なかったものが生まれてうまく動く瞬間にやりがいを感じるPM — ただし、細分化されたステークホルダーの調整とスケジュール管理が最大の課題だ。

| 質問 | 回答 |
|------|------|
| 今やっている仕事について簡単に教えてもらえますか？ | プロジェクト管理マネージャーです。 |
| 主にどの分野のプロジェクトを担当していますか？ | チャネル系もありますし、基幹系や新規事業導入のプロジェクト、SI系のプロジェクトがほとんどです。 |
| その中で一番面白い、またはやりがいを感じるのはどんな種類のプロジェクトですか？ | 新しいシステムを導入したり、ビジネスが生まれてサービスがうまく運用されているときにやりがいを感じます。 |
| 印象に残っているプロジェクトはありますか？ | 一番最近のプロジェクトは「カードデコレーション」プロジェクトで、エディターを使って自分好みのデザインでカードプレートを作るプロジェクトでした。 |
| そのプロジェクトで「うまくいった」と感じた瞬間は具体的にいつでしたか？ | エラーなく画像がうまく生成されたときです。 |
| プロジェクトを進める中で最も難しかった点は何でしたか？ | 複数のステークホルダーと協力し、対応開発が必要だったため、スケジュール管理が大変でした。 |
| 特にどのステークホルダーとの間で衝突することが多かったですか？ | チャネル側の担当者が細かい業務ごとに分かれていたので大変でした。 |
| そのときご自身なりの解決方法はありましたか？ | *(未回答)* |

## 2. AI活用への期待と消費習慣

**一言まとめ:** 意志より仕組みが必要な人 — わかっていても忘れてしまうため、AIが繰り返し気づかせてくれるシステムの役割を果たしてほしいと期待している。

| 質問 | 回答 |
|------|------|
| このインタビューを始めたきっかけは何ですか？ | 研修でやるように言われたからです。 |
| 言われて始めたけれど、実際にやってみて意味がありそうだと感じましたか？ | 意味があるとは思うんですが、まだ不慣れなので実際にうまく使えるかどうかはわかりません。 |
| どんな面で意味があると感じましたか？ | うまく使えば仕事や日常生活でとても役に立てると思います。 |
| 特に「こんな場面でAIがあればよかった」と感じた瞬間はありましたか？ | 旅行の日程を組んだり、カードの使用や毎月の家計簿を自動で作成して、節約すべき部分を整理して教えてくれればいいなと思います。 |
| 家計簿の面で特に気になる点はありますか？ | 習慣的なコーヒー代や、過剰なサブスクリプションサービスの費用です。 |
| わかっていても続けてしまう理由はありますか？ | 面倒くさくて解約できないんです。コーヒーはカプセルで淹れればいいのに、つい高いものを買ってしまう習慣があるようです。 |
| 高いコーヒーを買うのは味のためですか、それとも別の理由がありますか？ | 近くにあるからです。それと、誰かに買うときは低価格より有名ブランドのコーヒーを選ぶ傾向があります。 |
| 誰かに買う際に低価格のものは避けたいほうですか？ | そうです。 |
| それがご自身の支出にも影響していると思いますか？ | コーヒー代が月10万ウォン以上かかっています。毎月の消費金額をまとめていると、こんなにたくさん飲んでいたのかと驚くことがあります。 |
| 自分で気づくだけでは足りず、外部から継続的に意識させてもらう必要があると感じますか？ | そうです。ずっと気づかせてくれるといいですね。 |

## 総合インサイト

- **職業:** SI/チャネル分野のPM。新しいシステムが安定的に稼働するときにやりがいを感じ、多くのステークホルダーの調整とスケジュール管理が主な課題。
- **AIへの期待:** 家計簿の自動化、旅行日程の作成など、繰り返しの情報整理と支出警告の役割をAIに期待。意志よりも仕組み的なナッジ（通知）を好む。
