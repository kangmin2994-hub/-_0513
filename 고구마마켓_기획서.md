## 1. 서비스 개요

| 항목 | 내용 |
|------|------|
| **서비스명** | 고구마마켓 |
| **서비스 유형** | 위치 기반 중고거래 플랫폼 |
| **한 줄 소개** | 내 동네 이웃과 직접 연결되는 따뜻한 중고거래 서비스 |
| **핵심 가치** | 신뢰 · 편리함 · 지역 커뮤니티 |

### 서비스 배경

기존 중고거래 플랫폼은 원거리 거래 시 배송비 부담, 사기 피해, 소통 불편 등의 문제가 있습니다.
고구마마켓은 **위치 기반 필터링**으로 내 근처 판매자와 직접 만나 거래하여 이러한 문제를 해결합니다.

## 2. 타겟 사용자

### 주요 타겟
- **판매자:** 집에 있는 중고 물품을 빠르게 처분하고 싶은 사람
- **구매자:** 저렴하고 가까운 곳에서 물건을 구하고 싶은 사람

### 페르소나

| 구분 | 판매자 페르소나 | 구매자 페르소나 |
|------|----------------|----------------|
| **나이** | 30대 직장인 | 20~40대 |
| **상황** | 이사 후 짐 정리 필요 | 신혼·이사로 생활용품 필요 |
| **니즈** | 빠른 처분, 직거래 선호 | 저렴하고 상태 좋은 물건 |
| **불편함** | 원거리 구매자 응대 번거로움 | 택배 사기, 긴 배송 대기 |

## 3. 핵심 기능

### 3-1. 상품 등록 / 판매

- 사진 최대 10장 업로드
- 카테고리 선택 (의류, 가전, 가구, 도서, 식물 등)
- 가격 입력 및 **가격 제안 허용 여부** 설정
- 상품 상태 선택 (미사용 / 거의 새것 / 사용감 있음)
- 거래 방식 선택 (직거래 / 택배 / 둘 다 가능)

### 3-2. 채팅 / 쪽지

- 상품 페이지에서 **1:1 채팅** 바로 시작
- 채팅방 내 거래 예약 확정 기능
- 읽음 확인 및 알림 설정
- 거래 완료 후 **후기 남기기** 연동

### 3-3. 위치 기반 거래

- 현재 위치 또는 설정 동네 기준 반경 필터 (1km / 3km / 5km / 10km)
- 지도 뷰에서 근처 상품 한눈에 확인
- **직거래 장소 추천** (편의점, 지하철역, 공공장소)
- 동네 인증으로 신뢰도 강화

## 4. 서비스 플로우

### 판매자 플로우

```
회원가입 / 로그인
    ↓
동네 인증 (GPS 또는 주소 입력)
    ↓
상품 등록 (사진 · 제목 · 가격 · 카테고리)
    ↓
채팅 응대 (구매 희망자와 협의)
    ↓
거래 장소 · 시간 확정
    ↓
직거래 완료 → 거래 완료 처리
    ↓
후기 작성
```

### 구매자 플로우

```
회원가입 / 로그인
    ↓
동네 설정 → 근처 상품 탐색
    ↓
상품 상세 확인 (사진 · 설명 · 판매자 신뢰도)
    ↓
채팅으로 문의 / 가격 제안
    ↓
거래 장소 · 시간 협의
    ↓
직거래 완료 → 후기 작성
```

## 5. 화면 구성

| 화면 | 주요 내용 |
|------|-----------|
| **홈 화면** | 동네 설정 배너, 카테고리 탭, 근처 최신 상품 목록 |
| **상품 목록** | 거리순/최신순 정렬, 카테고리·거래방식 필터 |
| **상품 상세** | 상품 사진 슬라이드, 판매자 정보, 채팅하기 버튼 |
| **채팅 목록** | 진행 중인 채팅방, 안 읽은 메시지 배지 |
| **채팅방** | 1:1 메시지, 거래 예약 버튼, 장소 공유 |
| **상품 등록** | 사진 업로드, 카테고리·가격·설명 입력 |
| **마이페이지** | 판매 내역, 구매 내역, 관심 목록, 후기 확인 |
| **지도 뷰** | 내 위치 중심 상품 핀 표시 |

## 6. 기대 효과

### 사용자 측면
- 가까운 이웃과 직접 거래로 **배송비 절감**
- 실물 확인 후 거래로 **사기 피해 최소화**
- 동네 기반 신뢰 거래로 **편리한 중고 거래 경험**

### 비즈니스 측면
- 지역 커뮤니티 활성화로 **재방문율 향상**
- 거래 완료 후 후기 시스템으로 **신뢰도 축적**
- 위치 기반 광고 및 프리미엄 노출 **수익 모델 확장** 가능
